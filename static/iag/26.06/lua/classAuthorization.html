Detailed Description

This module allows you to set the results of a custom authorization decision. The functions within this object are only valid for 'preazn' rules.

◆ setDecision()

nil Authorization::setDecision ( string  decision,
boolean  audit,
table  attributes 
)

Used to set the authorization decision for the request. This function should only be called as a part of a 'preazn' rule.

Parameters
decision: The authorization decision. One of the following: 'deny' : The request is not authorized. 'allow' : The request is allowed without additional authorization checks. 'continue' : The request is allowed, but the native authorization check will still take place on this request.
audit: Should the decision be audited? If no value is specified the decision will not be audited. The auditing takes place using the audit.azn auditing component, and so the auditing record will only be generated if the standard auditing for authorization decisions has been enabled.
attributes: An optional table of attributes (name/value pairs) which will be included in the auditing record.
Return values
nil
Examples
Authorization.setDecision("allow")
Authorization.setDecision("allow", false)
Authorization.setDecision("allow", true)
Authorization.setDecision("allow", true, {
["attr_1"] = "value_1",
["attr_2"] = "value_2
})
