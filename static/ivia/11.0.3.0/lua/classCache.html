Detailed Description

This module allows you to cache data for a particular rule. This data is available to all requests which are processed by this rule. The data is stored in an expiring LRU cache. The default maximum size of the cache is set to 8192 records.

◆ setMaxSize()

nil Cache::setMaxSize ( number  maxSize)

Used to set the maximum size of the cache. Once the cache reaches the maximum size the oldest entries will be removed to make way for the newest entries (ie a least-recently-used algorithm will be used). By default, the cache will be initialised with a maximum size of 8192 entries.

Parameters
maxSizeThe maximum number of entries to be stored in the cache.
Return values
nil
Examples
Cache.setMaxSize(1024)
