Lua Module Documentation

This is the documentation for the custom Lua modules which can be used when writing HTTP transformation rules for IBM Application Gateway.

Note
Details on the syntax of the Lua scripting language can be found in the official Lua documentation: https://www.lua.org/manual/5.4/

In addition to the custom Lua modules the following LuaRocks modules have been preinstalled:

  • basexx
  • binaryheap
  • cqueues
  • fifo
  • http
  • lpeg
  • lpeg_patterns
  • lua-cjson
  • lualdap
  • luaossl
  • luasocket
  • redis-lua
  • urlencode
Warning
To help protect the appliance a number of functions from the standard Lua libraries have been disabled. This includes:
  • Basic
    • loadfile
  • Operating System Facilities
    • execute
    • exit
    • remove
    • setlocale
    • tmpname
  • Input and Output
    • all functions
  • Package
    • loadlib
